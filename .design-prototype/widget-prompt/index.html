Direção 01

Code Block

Bloco de código estilo GitHub/Notion. Header com label de tipo + botão copiar. Mono, denso, familiar para quem já usa editores. Botão vira "Copiado ✓" por 2s.

Na Aula 06 você colocou uma tela Next.js pra rodar. Agora o agente precisa de um pedido bem formulado pra criar a lista de tarefas. Use o prompt abaixo:

Prompt · Claude Code 
Crie uma lista de tarefas simples neste app Next.js.
A lista deve ter: um campo de texto, um botão Adicionar e os
itens aparecem abaixo. Cada item tem um botão pra excluir.
Não instale nenhuma nova dependência, só use o que já tem.

Depois de copiar, abra o Claude Code no VS Code, clique na caixa de chat e cole com Ctrl+V (ou Cmd+V no Mac).
