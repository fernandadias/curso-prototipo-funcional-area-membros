Se você já tem uma conta no Claude (ou no Claude Code) e sabe qual plano vai usar, pode fechar essa aba e voltar pra aula. Isso aqui é pra quem está criando a conta pela primeira vez ou ainda não escolheu um plano.

## Os planos do Claude

O Claude Code é um produto da Anthropic — a mesma empresa que faz o Claude. Pra usar o Claude Code, você precisa de uma conta paga. Veja as opções:

Preços e condições mudam com frequência. Sempre confira os valores atuais em [claude.ai/upgrade](https://claude.ai/upgrade) antes de assinar.

<Callout tipo="atencao">
  **Para o curso:** o plano **Pro** é suficiente. Você vai usar o Claude Code em sessões de trabalho pontuais — não em produção contínua. Se em algum momento você atingir os limites durante uma sessão, o agente avisa e você retoma em alguns minutos ou no dia seguinte.
</Callout>

## Criando a conta: passo a passo

<ol className="steps">
  <li>**Acesse [claude.ai](https://claude.ai)** e clique em *Sign up*.</li>
  <li>**Escolha como criar a conta.** Pode usar Google, Apple ou e-mail. Qualquer uma funciona igual.</li>
  <li>**Confirme o e-mail** se escolheu entrar com e-mail. Procure a mensagem da Anthropic na caixa de entrada (ou no spam).</li>
  <li>**Escolha o plano.** Após criar a conta, vá em *Upgrade* (canto inferior esquerdo do chat) e assine o **Pro**.</li>
  <li>**Confirme o pagamento.** O acesso ao Claude Code é liberado imediatamente após a assinatura.</li>
  <li>**Pronto.** Na hora de instalar a extensão do Claude Code no VS Code (Aula 06), clique em *Login* e entre com essa mesma conta. Ele vai autenticar via navegador automaticamente.</li>
</ol>

<Callout tipo="fonte">
  **Fonte oficial:** [Página de planos do Claude](https://claude.ai/upgrade) — preços atuais e comparativo completo. É a fonte mais confiável; os valores acima são aproximados e podem ter mudado.
</Callout>
