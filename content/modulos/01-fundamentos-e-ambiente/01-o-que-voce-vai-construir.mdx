Antes de qualquer conceito ou instalação, quero te mostrar o que está te esperando
neste curso. Você vai construir dois projetos: um simples pra colocar o pé na água
e um brabo pra explorarmos o potencial de código pra fazer design.

## O projeto de treino: lista de tarefas

A ideia é simples, mas já com ela vamos aprender toda a base dos protótipos
funcionais: configurar o ambiente com IA, como iniciar um protótipo, ações de criar
e excluir itens, salvar dados… Os dois primeiros módulos têm 80% das melhores
práticas pra criar experiências funcionais incríveis. Depois só vamos colocar uns
Power Ups nelas 🍄

## O mega protótipo: Watchlist

A partir do módulo 4 a conversa fica séria. Depois de ter as bases na cabeça vamos
explorar um projeto de design mais robusto, com interações complexas para exercitar
como realmente é usar código no processo de design.

O projeto core do curso será **uma Watchlist**: um app pra gerenciar filmes e séries
que você quer ver, com dados reais dos filmes, banco de dados para salvar nossas
interações, feature de IA para gerar recomendação, deploy pra qualquer pessoa acessar
e mais umas paradas iradas que vou deixar pra você descobrir na hora certa.
